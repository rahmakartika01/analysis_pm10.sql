# analysis_pm10.sql
SQL project for analyzing PM10 air pollution trends and station comparisons.

## Overview
This project analyzes PM10 air pollution data during September 2024 – February 2025 using SQL.

## Tools
- PostgreSQL
- SQL
- Window Function
- CTE

## Analysis
- Data Cleaning
- Moving Average (7, 14, 30 days)
- PM10 aggregation per station
- Highest and lowest PM10 station analysis

## Key Insight
DKI4 (Lubang Buaya) has the highest PM10 average, while DKI5 (Kebon Jeruk) has the lowest average during the observation period.
